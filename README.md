# LOOP Client Portal

Portal web informativo para clientes de LOOP AI.

## Objetivo
Entregar a cada cliente una vista segura de su operacion (llamadas, citas, mensajes, comparativas y reportes), aislada por `clientCode`.

## Regla de separacion
Este repositorio es independiente de `loop-admin`.
No se debe modificar codigo de `loop-admin` desde aqui.

## Estado
Fase de planeacion (sin codigo funcional).

## Estructura
- `docs/` documentacion de producto y arquitectura.
- `specs/` definiciones funcionales detalladas por modulo.
- `assets/` recursos visuales y branding.
